### 阶段 1：选题定题 + 文献调研（4.10-4.12，3 天）**核心：贴合主题，拒绝空泛**

1. 每日任务

   - 4.10：全员围绕**服务国家战略 创新统计赋能** brainstorm，每人出 3 个选题（方向：数字经济、碳中和、乡村振兴、医疗健康、就业统计等）
   - 4.11：筛选 2 个最优选题，查知网 / 统计局公开数据**可获得性**，排除无数据题目
   - 4.12：最终定题，队长撰写选题背景，全员下载 10 篇核心参考文献

   

2. 硬性要求：选题小而精，**必须有公开可获取数据**，不做纯理论研究

3. 交付物：最终题目、参考文献包、选题背景初稿

### 阶段 2：数据采集 + 清洗预处理（4.13-4.15，3 天）**核心：数据可溯源，无缺失值**

1. 每日任务

   - 4.13：数据岗下载原始数据（国家统计局、地方统计局、 Kaggle、行业公开数据库）
   - 4.14：数据清洗（去重、补缺失值、异常值处理、指标标准化）
   - 4.15：建模岗对接数据，确认数据维度匹配模型，数据岗完成数据说明

   

2. 硬性要求：数据留存**原始版 + 清洗版**，标注来源，打包备用

3. 交付物：完整数据包（Excel/CSV）、数据来源说明文档

### 阶段 3：模型搭建 + 实证运算（4.16-4.19，4 天）**核心：模型适配，结果可复现**

1. 每日任务

   - 4.16：建模岗确定模型框架（描述统计 + 回归 / 聚类 / 机器学习等统计模型），写代码框架
   - 4.17：全员联调，跑通基础模型，输出初步结果
   - 4.18：模型优化（调参、稳健性检验），制作核心图表（折线图、热力图、回归表）
   - 4.19：固化模型结果，建模岗填写**AI 工具使用情况表**（无使用填无，严禁隐瞒）

   

2. 硬性要求：代码注释完整，结果可复现，图表统一格式

3. 交付物：模型代码、实证结果、可视化图表、AI 使用表初稿

### 阶段 4：论文撰写 + 格式合规（4.20-4.23，4 天）**核心：严控字符 + 查重，结构标准**

1. 论文固定结构（严格贴合大赛要求）：摘要→关键词→引言→数据说明→模型设计→实证分析→结论与建议→参考文献

2. 每日任务

   - 4.20：队长写摘要 / 引言 / 结论，数据岗写数据部分，建模岗写模型部分
   - 4.21：全员拼接论文，统一语言风格，补充图表标注
   - 4.22：精简内容，**严控正文字符≤16000（计空格）**，删除冗余表述
   - 4.23：格式排版（字体、行距、参考文献格式），完成论文终稿初稿

   

3. 硬性要求：**原创性表述≥80%**，不抄袭、不套用往届论文

4. 交付物：格式规范的论文初稿（字符达标）
