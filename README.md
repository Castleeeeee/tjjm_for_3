## 项目说明

本仓库用于2026年统计建模大赛的3天极限速成准备，提供完整的参赛流程指导和参考资料。

## 目录说明

- **讲义PDF**：包含大赛指导、时间规划、各阶段任务分配等核心内容
- **demo.md**：详细6阶段参赛流程指南，涵盖选题定题、数据采集、模型搭建、论文撰写、合规校验到最终提交的完整流程

## 参赛流程（6阶段）

1. **选题定题 + 文献调研**（4.10-4.12，3天）：围绕"服务国家战略 创新统计赋能"主题确定题目
2. **数据采集 + 清洗预处理**（4.13-4.15，3天）：采集公开数据并进行清洗处理
3. **模型搭建 + 实证运算**（4.16-4.19，4天）：搭建统计模型并进行实证分析
4. **论文撰写 + 格式合规**（4.20-4.23，4天）：撰写符合大赛要求的论文
5. **合规校验 + 全套材料打包**（4.24-4.25，2天）：查重、校验、打包提交材料
6. **最终提交 + 备份存档**（4.26，截止16:00）：提交参赛材料并备份
