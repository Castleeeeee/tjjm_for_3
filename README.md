# tjjm_for_3

2026统计建模大赛参赛项目

## 项目简介

本仓库用于2026年统计建模大赛的准备与参赛，提供完整的参赛流程指导、参考资料和数据分析代码。

## 目录结构

```
d:\tjjm_for_3\
├── reference/                # 参考资料目录
│   ├── 工具集/                # 包含优化模型、评价模型、神经网络等工具集
│   ├──往届资料/               # 包含往届参赛的资料
│   └──指导书/                 # 包含讲义、模版、选题分析等
├── data_analysis/            # 数据分析目录                     
├── README.md                 # 项目说明文档
└── demo.md                   # 6阶段参赛流程指南
```

## 参赛流程（6阶段）

### 1. 选题定题 + 文献调研（4.10-4.12，3天）
围绕"服务国家战略 创新统计赋能"主题确定题目，每人提出3个选题方向，筛选后确定最终题目。

### 2. 数据采集 + 清洗预处理（4.13-4.15，3天）
从国家统计局、地方统计局、Kaggle等公开数据源采集数据，进行清洗和预处理。

### 3. 模型搭建 + 实证运算（4.16-4.19，4天）
搭建统计模型（描述统计、回归、聚类、机器学习等），进行实证分析和结果可视化。

### 4. 论文撰写 + 格式合规（4.20-4.23，4天）
撰写符合大赛要求的论文，严格控制字符数≤16000，确保原创性≥80%。

### 5. 合规校验 + 全套材料打包（4.24-4.25，2天）
查重（目标≤20%）、校验所有材料完整性，打包提交材料。

### 6. 最终提交 + 备份存档（4.26，截止16:00）
提交参赛材料并进行双备份（云盘+本地）。

## 必交材料

- 参赛报名表
- 参赛承诺书
- AI工具使用情况表
- 参赛论文（终稿）
- 知网查重报告
- 数据包（原始+清洗版+说明）

## 使用说明

- `demo.md` 包含详细的每日任务分解和硬性要求
- `reference/` 目录存放参考文献和参考资料
- `data_analysis/` 目录存放所有数据分析相关文件
